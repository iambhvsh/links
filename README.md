# Links

A central place for work, writing and everything else.

Designed as a lightweight and editorial-style links page focused on clarity, typography and thoughtful interaction.

---

## Overview

This project acts as a minimal personal directory for navigating across projects, platforms and digital spaces.

Rather than functioning as a traditional social links page, the experience is designed to feel calm, structured and intentional — presenting links as part of a cohesive interface system rather than a collection of buttons.

The page serves as an entry point into the broader ecosystem.

---

## Philosophy

The design is guided by a few consistent principles:

- Clear visual hierarchy
- Editorial-inspired layout
- Refined interaction feedback
- Comfortable spacing and rhythm
- Lightweight performance
- Focused browsing experience

The overall direction draws inspiration from modern product interfaces, publishing systems and minimal digital environments.

---

## Features

### Interface

- Responsive layout across devices
- Automatic light and dark appearance support
- Smooth and subtle transitions
- Accessible keyboard interactions
- Typography-focused design system
- Soft elevation and layered surfaces
- Minimal and distraction-free structure

### Technical

- Single-file architecture
- Semantic HTML structure
- Optimized rendering performance
- SEO and Open Graph support
- Lightweight asset footprint
- Modern browser compatibility

## Sections

The page includes links to:

- Main website
- Work
- Writing
- GitHub
- LinkedIn
- Instagram
- Résumé
- Email

Each destination is presented using a consistent card-based interaction system.

---

## Tech Stack

Built using a lightweight frontend stack focused on simplicity and performance.

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
- Material Symbols

No frameworks are used.

---

## Design System

The interface is built around a custom visual system including:

- 8px spacing grid
- Adaptive theme tokens
- Rounded geometry
- Editorial typography
- Soft shadows and borders
- Refined motion curves
- Consistent interaction states

The result is a links page that feels closer to a product interface than a traditional link aggregator.

---

## Accessibility

Accessibility considerations include:

- Semantic markup
- Keyboard navigation support
- Visible focus states
- Reduced motion handling
- Responsive scaling
- Readable contrast ratios

---

## Performance

The page is optimized for fast loading and smooth interaction across modern devices.

Minimal JavaScript and lightweight assets help maintain a responsive experience while preserving visual detail.
